## Software needed to replicate & contents
- Hyprland, Hyprlock, Hypridle
- wal
- swww
- rofi
- waybar
- kitty
- nvim
- swaync

### No-Conf Software:
- ly (kept stock)


## Usage

1. clone into $HOME (~/) & cd  
2. install stow && run `stow .` to symlink files into .config
